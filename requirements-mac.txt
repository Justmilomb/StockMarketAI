# Mac desktop terminal requirements (PySide6 .app + Claude agent + full ML ensemble).
#
# Install:
#   python3 -m venv .venv-mac
#   source .venv-mac/bin/activate
#   pip install -r requirements-mac.txt
#
# This is the macOS sibling of requirements-desktop.txt. Same Python
# packages — PySide6 / pyqtgraph / pyinstaller all support macOS as a
# first-class target. Differences vs requirements-desktop.txt:
#
#   * No pywin32 / Windows-only PyInstaller hooks (none are listed in
#     requirements-desktop.txt today, but if any are added there they
#     should NOT be added here without checking macOS support first).
#   * pyobjc-* is added so PyInstaller can build an .app bundle that
#     correctly registers with macOS launch services and shows up in
#     the Dock with the right icon. PyInstaller pulls these via
#     hidden imports anyway — we list them here so a fresh ``pip
#     install`` doesn't surface a build-time NameError.
