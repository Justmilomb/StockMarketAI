## 2. 🧩 Add *regime switching* (this is huge)

Right now:

> You only trade **mean reversion**

That’s fragile.

### Upgrade:

Instead of:

```
if regime == mean_reverting → trade
```

Do:

```
if regime == mean_reverting → strategy A
if regime == trending → strategy B
if regime == high vol → reduce size / no trade
```

👉 This is how you:

* smooth equity curve
* avoid getting destroyed in wrong environments

## 3. ⚖️ Make position sizing *adaptive*

Your current:

* fixed 12% per position

### Upgrade to:

* signal strength
* volatility
* correlation to other positions

```
size ∝ (prediction confidence) / (volatility * correlation)
```

👉 This alone can:

* increase Sharpe
